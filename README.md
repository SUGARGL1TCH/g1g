# g1g

G1G is the codename for a visual novel project by Tit@, 
The game is developed with javascript using the phaser library and can be played in a browser

The game can be found at http://www.gamacore.com/g1g

# Plot
the game is about a teenager who was daydreaming during math class out of bordom, she then enters an interctive dream.
the structure of the game is described in the file structure.yaml
